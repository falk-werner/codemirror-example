# CodeMirror-Example

This repository contains a brief example how to integrate
[Code-Mirror](https://codemirror.net/) using [vite](https://vitejs.dev/).

## Try using developer mode

```bash
npm run dev
```

## Create standalone web application

```bash
npm run build
```
